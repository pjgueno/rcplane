# RC plane

A RC plane made with the Arduino IDE:

Remote:
- screen ST7735
- 2 joystick potentiometers
- 1 slider for throttle
- nRF24L01 2.4GHz Transceiver
- Arduino Mega

Plane:
- 5 servos
- 1 accelerometer MPU-6050
- 1 ESC with motor
- nRF24L01 2.4GHz Transceiver
- ESP32

The screen shows an artificial horizon.

RF24 library version V1.1.7
